DCA 월복리

연 수익률을 월 유효수익률로 변환한 뒤 기존 잔액에 적용하고 월말 투자금을 더합니다. 연도 경계에서 투자금 증가율을 반영합니다.

월 수익률 = (1 + 연 수익률)^(1/12) - 1

하락 후 회복률

하락 후 남은 비율의 역수를 사용합니다. 50% 하락 뒤 원금을 회복하려면 50%가 아니라 100% 상승해야 합니다.

회복률 = 1 / (1 - 하락률) - 1

스트레스 테스트

자산별 보유 금액에 선택하거나 수정한 충격률을 곱해 시나리오 후 금액과 손익을 합산합니다.

시나리오 금액 = Σ 보유 금액 × (1 + 충격률)

포트폴리오 위험 점수

상위 자산 집중도, 레버리지·기술 성장군·코인 노출은 점수를 높이고 현금·채권 노출과 자산 수에 따른 분산도는 완충 요소로 반영합니다.

레버리지 일간 복리

매일 생성된 기초자산 수익률에 선택 배율을 적용하고 일별로 복리 누적합니다. 단순 배율 값은 전체 수익률에 배율을 한 번 적용합니다.

일별 잔액 = 이전 잔액 × (1 + 배율 × 일간 수익률)

리밸런싱

최종 총자산에 목표 비중을 곱한 금액과 현재 자산 금액의 차이를 조정 금액으로 사용합니다.

목표 금액 = (현재 총액 + 추가금) × 목표 비중

리밸런싱 계산기 열기

FIRE 목표 금액

직접 입력한 목표가 없으면 현재 기준 연 생활비를 인출률로 나눕니다. 미래 목표는 물가상승률에 따라 증가합니다.

현재 목표 금액 = 연 생활비 / 인출률

FIRE 계산기 열기

통화 환산

원화 입력을 선택하면 입력 금액을 환율로 나눠 내부 기준 금액으로 계산한 뒤 결과를 다시 원화로 표시합니다. 환율은 사용자가 직접 입력한 고정 가정입니다.

달러 기준 금액 = 원화 입력 금액 / 입력 환율
